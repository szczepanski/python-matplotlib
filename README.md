# python-matplotlib

## Basic Graph

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [6, 7, 4, 5, 2]

plt.plot(x, y)
plt.title('Basic Graph \n'
          'with a new line subtitle')
plt.xlabel('x axis label')
plt.ylabel('y axis label')
plt.show()
```
